共享管理UI详细设计

日期 版本号 内容摘要 作者
2017.08.11 共享管理UI详细设计

##1 概述
本文档主要针对共享管理进行详细设计说明

###1.1 编写目的
本文档主要为代码开发做准备，是对共享管理需求文档和设计文档的总结，帮助整
理开发思路、明确开发步骤、指导开发有效的进行。

###1.2 参考资料

  • YeeSAN1.x需求说明

###1.3 名词解释

##2 背景说明

###2.1 框架背景
使用Backbone框架进行开发

###2.2 项目背景

###2.3 运行环境
Chrome Browser

##3 模块设计
统一管理系统中的共享目录，支持CIFS和NFS两种共享方式，可以支持创建新的共享目录，删除共享目录，目录的访问权限管理。

###3.1 NFS共享
NFS共享管理页面，主要管理系统中所有支持NFS共享的目录，支持包括创建NFS共享目录操作，删除NFS共享目录，管理NFS共享目录的客户端信息

#####文件基本结构
view(视图文件):
/YeeSAN-GUI/frontend/js/views/sysconfig/shareManage/ShareManage.js（共享管理）
/YeeSAN-GUI/frontend/js/views/sysconfig/shareManage/NfsShare.js（NFS共享）
/YeeSAN-GUI/frontend/js/views/sysconfig/shareManage/NfsShareClientInfo.js（NFS共享客户端信息）
/YeeSAN-GUI/frontend/js/views/sysconfig/shareManage/AddNfsShare.js（创建NFS共享）
/YeeSAN-GUI/frontend/js/views/sysconfig/shareManage/DirectoryScan.js（文件目录浏览）

####3.1.1 NFS共享列表
NFS共享目录列表信息如下：

  1. 共享路径： 共享的目录的路径名称
  2. 描述： 共享目录的描述信息

获取NFS共享流程图如下

####3.1.2 创建NFS共享
点击创建按钮，弹出创建NFS共享的弹出框，输入相应信息，点击确定按钮，向服务器提交新创建NFS共享的信息，弹窗关闭，NFS共享列表刷新

创建NFS共享流程图如下

####3.1.3 删除NFS共享
选中要删除的NFS共享目录，点击删除按钮，弹窗删除NFS共享的弹窗框
删除NFS流程

####3.1.4 客户端信息
选中要查看客户端信息的NFS共享，点击“客户端信息”按钮，弹窗客户端信息弹框

客户端信息类别包括以下几点：

  1. 业务IP地址：即可以访问该共享目录的客户端的IP地址，可以输入固定IP地址或者*
  2. 读写模式：异步、同步两种模式
  3. 读写权限：读写、只读两种权限

客户端信息支持添加、修改、删除三种操作，所有的操作如果不点击客户端信息弹框的确定按钮，都不会将信息发往服务端

添加客户端流程

####3.1.5 搜索NFS共享
提供搜索NFS共享功能，在搜索框输入要搜索NFS的共享的共享路径，目前不支持模糊搜索

###3.2 CIFS共享
NFS共享管理页面，主要管理系统中所有支持CIFS共享的目录，支持包括创建CIFS共享目录操作，删除CIFS共享目录，管理CIFS共享目录的用户/用户组信息

####文件基本结构

view(视图文件):
/YeeSAN-GUI/frontend/js/views/sysconfig/shareManage/ShareManage.js（共享管理）
/YeeSAN-GUI/frontend/js/views/sysconfig/shareManage/CifsShare.js（CIFS共享）
/YeeSAN-GUI/frontend/js/views/sysconfig/shareManage/CifsShareUserInfo.js（CIFS共享用户/用户组）
/YeeSAN-GUI/frontend/js/views/sysconfig/shareManage/AddCifsShare.js（创建CIFS共享）
/YeeSAN-GUI/frontend/js/views/sysconfig/shareManage/DirectoryScan.js（文件目录浏览）

####3.2.1 CIFS共享列表
CIFS共享目录列表信息介绍如下:

  1. 共享名称：CIFS共享时的该目录的别名
  2. 共享路径：该目录的共享路径
  3. 描述：共享目录的描述信息
    ####3.2.2 创建CIFS共享
    点击“创建”按钮，弹出创建CIFS共享的弹框

创建CIFS共享流程图如下

####3.1.3 删除CIFS共享
选中要删除的NFS共享目录，点击删除按钮，弹窗删除NFS共享的弹窗框
删除CIFS流程

####3.2.1 用户/用户组信息

选中要查看客户端信息的CIFS共享，点击“客户端信息”按钮，弹窗用户/用户组信息弹框

  1. 用户/用户组:支持两种类型，域用户、域用户组；
  2. 读写权限：支持读写、只读两种权限，默认为只读权限
  3. 创建的共享名称不能与已有的共享名称重复,如果重复，则提示用户“输入共享名称和已有共享名称重复，请重新输入”
  4. 如果系统接入AD域，创建共享目录，默认所有AD域用户不可访问共享该目录，可以单独设置域用户/域用户组的的访问权限；
  5. 如果系统未接入AD域，创建共享目录，默认所有的用户都可以访问共享该目录；

用户/用户组信息支持添加、修改、删除三种操作，所有的操作如果不点击用户/用户组信息弹框的确定按钮，都不会将信息发往服务端

添加用户/用户组流程

#对象存储UI详细设计文档

日期 版本号 内容摘要 作者
2017.04.17 对象存储UI详细设计文档 崔德奇

##1. 概述
本文档主要针对对象存储项目进行详细设计说明

###1.1 编写目的
本文档主要为代码开发做准备，是对对象存储需求文档和设计文档的总结，帮助整
理开发思路、明确开发步骤、指导开发有效的进行。

###1.2 参考资料

  • 日志汇聚系统项目需求文档
  • 对象存储交互及界面设计文档(TODO)

###1.3 名词解释

##2. 背景说明

###2.1 框架背景
使用Backbone框架进行开发

###2.2 项目背景

###2.3 运行环境
Chrome Browser

##3. 模块设计
在Woostor的数据中心页面新增一个导航栏日志管理。日志管理下面包含日志统计、日志操作、策略配置个模块。

###3.1 日志统计模块
日志统计模块包含服务统计、数据统计组件

####3.1.1 服务统计组件
服务统计组件包含对象服务、网关服务、存储服务子组件

#####功能概要
显示对象服务、网关服务、存储服务的信息。

  1. 对象服务，即当前数据中心的对象存储的状态。
  2. 网关服务，即当前数据中心得到网关服务状态。
  3. 存储服务，即当前数据中心的OSS服务的状态。
  4. 服务状态有两种，包括正常、异常的两种状态，同时分别统计处于两种状态的对象服务个数。
    #####文件基本结构
    view(视图文件):
    ServiceStatistics.js（服务统计视图）
    LogStatistics（日志统计视图）

hbs(模板文件):
serviceStatistics.hbs（服务统计模板）
serviceStatisticsLegend.hbs（π图图例模板）

因为不涉及数据的频繁变化，只显示数据，所以基本的数据操作都放在视图中进行，没有建立model进行数据的转换。

#####组件接口
路由跳转到该模块自动触发视图加载

#####流程图

####3.1.2 数据统计组件

#####功能概要

  1. 以条状图的形式显示过去一周的日志条数和日志容量，包括统计数据的当天。
  2. 日志条数以万条为单位，日志容量以GB为单位。
  3. 单击树状图，可以显示或者隐藏改类数据的显示
  4. 刷新频率：60秒

#####文件基本结构
